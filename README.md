# ConversIA

ConversIA is a modern web application designed to manage client conversations and train AI chatbot responses in an intuitive and structured way.

Built with a scalable architecture using React, Zustand, React Router, TailwindCSS, and Chart.js.

---

## 🚀 Tech Stack

- ⚛️ React 19
- ⚡ Vite
- 🎨 TailwindCSS
- 🧠 Zustand (Global State Management)
- 🗺 React Router
- 📊 Chart.js + react-chartjs-2
- 🧹 ESLint

---

## 📌 Features

- 📋 Conversation list view
- 💬 Chat simulation interface
- 🤖 Bot response training panel
- 📦 Client product management section
- 📊 Data visualization with charts
- 🧠 Global state management with Zustand
- 📱 Mobile-style UI layout
